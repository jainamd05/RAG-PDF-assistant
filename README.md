# 📚 PDF RAG Assistant using Gemini, LangChain & Qdrant
A Retrieval-Augmented Generation (RAG) application that allows users to ask questions from PDF documents and receive context-aware answers generated by Google's Gemini model.

## 🚀 Features

- PDF document ingestion
- Automatic text chunking
- Sentence embeddings using HuggingFace
- Vector storage with Qdrant
- Semantic similarity search
- Context-aware answer generation using Gemini
- Source page references in responses

## 🏗️ Architecture

PDF Documents -> Document Loader -> Text Chunking -> Embeddings (all-MiniLM-L6-v2) -> Qdrant Vector Database -> Similarity Search -> Retrieved Context -> Gemini 2.5 Flash
      
Generated Answer

## 🛠️ Tech Stack

- Python
- LangChain
- HuggingFace Embeddings
- Qdrant Vector Database
- Google Gemini 2.5 Flash
- Docker

## 📂 Project Workflow

### 1. Indexing

The indexing pipeline:

- Loads PDF documents
- Splits documents into chunks
- Generates vector embeddings
- Stores embeddings in Qdrant

Run:

```bash
python index.py
```

### 2. Question Answering

The chat pipeline:

- Accepts user query
- Converts query into embeddings
- Retrieves relevant chunks from Qdrant
- Sends context + question to Gemini
- Generates grounded response

Run:

```bash
python chat.py
```

## ⚙️ Setup

### Clone Repository

```bash
git clone https://github.com/yourusername/RAG-PDF-Assistant.git
cd RAG-PDF-Assistant
```

### Create Virtual Environment

```bash
python -m venv venv
```
Activate the virtual environment:

For Windows:
```bash
venv\Scripts\activate
```

For Linux/Mac
```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## 🔑 Environment Variables

Create a `.env` file
```env
API_KEY=YOUR_GEMINI_API_KEY
```


## 🐳 Run Qdrant
```bash
docker-compose up -d
```

Qdrant Dashboard:
```text
http://localhost:6333/dashboard
```
